# 道路工程品質計畫規格（金質獎 11 章制）

## 一、章節架構（第一章～第十一章）

| 章次 | 章名 | 說明 |
|-----|------|------|
| 第一章 | 計畫範圍 | 依據/工程概要/施工項目/適用對象/名詞定義 |
| 第二章 | 管理權責及分工 | 品管組織/工作職掌/人員輪值/管理審查/表單 |
| 第三章 | 施工要領 | 各工項8節詳述格式 |
| 第四章 | 品質管理標準 | 表四-1-1～四-1-N（各工項） |
| 第五章 | 材料與設備及施工檢驗程序 | 含材料自主檢查表（EM-XX）20份 |
| 第六章 | 設備功能運轉檢測程序及標準 | 含機電設備功能運轉檢測 |
| 第七章 | 自主檢查表 | 每工項3張（施工前/中/後），共N×3張 |
| 第八章 | 不合格品之管制 | 流程圖＋表格 |
| 第九章 | 矯正與預防措施 | 獨立章（景觀無此章） |
| 第十章 | 內部品質稽核 | 獨立章，5張表（景觀無此章） |
| 第十一章 | 文件紀錄管理系統 | 收發文流程＋文件分類代碼 |

> ★ 道路工程章次用「第X章」阿拉伯數字，非景觀的壹貳參中文章次。
> ★ 道路工程有獨立矯正預防章（第九章）和稽核章（第十章），景觀無。

## 二、各章製作要點

### 第一章 計畫範圍

- **依據（一節）**：工程契約、技師法、建築法、營造業法、施工品質管理制度
- **工程概要（二節）**：直接沿用監造計畫工程概要，新增承攬廠商欄
- **表一-1 工程主要施工項目及數量**（欄位：項次｜說明｜單位｜數量）
- **適用對象（三節）**：列明本計畫適用範圍
- **名詞定義（四節）**：20項以上，含「檢驗停留點（限止點）」、「安衛查驗點◎」

### 第二章 管理權責及分工

- **圖二-1 品管組織架構圖**（PIL PNG）：管理階層→工地負責人→品管/現場/行政/安全衛生→協力廠商
- **表二-1 工作職掌表**：職稱｜工作職掌及依據
- **人員輪值**：依勞基法施行細則第23條之1規定
- **管理審查**：每季審查，含稽核結果、矯正/預防追蹤
- **表二-2 督察紀錄表**（EP-）
- **表二-3 施工日誌**（道路工程特有）

### 第三章 施工要領（8節詳述格式）

每工項包含8節：
1. 工項概述（概要、內容、數量）
2. 人員組織
3. 施工方法與步驟
4. 施工機具（名稱/規格/數量）
5. 使用材料
6. 預定作業進度
7. **分項品質計畫**（含施工要領、品質管理標準、材料與設備及施工檢驗程序、自主檢查表）
8. 分項作業安全衛生管理

> 道路工程施工要領用8節詳述，而非景觀的4欄表格。

**表三-1 各分項計畫管制表**：項次｜提送項目｜預定送審日期｜備註

### 第四章 品質管理標準

**表四-1-X 各工項品質管理標準表**（欄位同景觀，但加入安衛停留點）：

| 施工流程 | 管理項目 | 管理標準 | 檢查時機 | 檢查方法 | 檢查頻率 | 管理紀錄 | 不符合標準之處置方法 |

**道路工程關鍵量化數值**：
- 瀝青混凝土壓實度：≧96%（景觀是95%）
- 瀝青鋪設時材料溫度：≥120°C
- 瀝青平整度：標準差S ≦0.26cm
- 黏層（MC-70）噴灑量：0.9～1.4 L/m²
- 混凝土坍度：15±3.5cm（景觀是±4cm）
- 混凝土氯離子：≦0.15 kg/m³
- 混凝土運送：出廠至澆置完成不超過90分鐘
- 試體：每20~100m³取1組，超過100m³每增加100m³加取1組

### 第五章 材料與設備及施工檢驗程序

### 第七章 自主檢查表（道路金質獎特色：每工項3張）

每工項3張自主檢查表：

- 施工中自主檢查表（EW-XX-2）
- 施工完成自主檢查表（EW-XX-3）

**停留點標記**：☆ = 需向監造申請會同查驗的項目（標記於檢查項目欄左側）

**底部固定格式**（道路加強版）：
```
缺失複查結果：
□已完成改善（檢附改善前中後照片）
□未完成改善，填具「缺失改善追蹤表」進行追蹤改善

複查日期：  年  月  日　複查人員職稱：       簽名：

備註：
1.☆表示檢驗停留點之檢驗項目。
2.檢查標準應具體明確或量化尺寸。
3.合格「○」，不合格「╳」，不需檢查「／」。
4.嚴重缺失未及時改善，應填具「不合格品管制總表」追蹤改善。
5.本表由工地現場施工人員實地檢查後覈實記載簽認。

工地主任（工地負責人）簽名：＿＿＿＿　　現場施工人員（檢查人員）簽名：＿＿＿＿
```

## 三、道路工程 vs 景觀工程關鍵差異

| 項目 | 道路工程（11章） | 景觀工程（8/9章） |
|-----|--------------|--------------|
| 章次格式 | 第X章（數字） | 壹貳參（中文） |
| 施工要領 | 8節詳述 | 4欄表格 |
| 停留點符號 | ☆（自主）＋◎（安衛） | ＊（監造）＋※（自主） |
| 自主檢查表 | 每工項3張（前/中/後） | 每工項1張 |
| 材料自主檢查 | 有（第五章 EM-XX） | 無 |
| 矯正預防章 | 有（第九章獨立） | 無（整合入不合格品章） |
| 稽核章 | 有（第十章獨立，5張表） | 無 |
| 缺失改善複查 | 加「改善前中後照片」 | 標準格式 |
| 混凝土坍度 | 15±3.5cm | 15±4cm |
| 施工日誌 | 有（表二-3） | 無 |

### 表名與圖名格式

- 表名：位於表格**上方**，水平置中，格式 `表{章號}-{序號} 表名`
  - 例：`表1-1 工程主要施工項目及數量表`、`表2-1 工作職掌表`
- 圖名：位於圖形**下方**，水平置中，格式 `圖{章號}-{序號} 圖名`
  - 例：`圖2-1 品管組織架構圖`、`圖5-1 材料/設備選定送審流程圖`
- 程式碼：使用 `add_table_caption(doc, chapter, seq, name)` 與 `add_figure_caption(doc, chapter, seq, name)`（見 `06_docx_tech.md` §八）
