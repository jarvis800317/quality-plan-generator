# 監造計畫 ⟷ 品質計畫 對應關係

> 適用：景觀工程9章制（壹貳參…玖）
> 道路工程11章制見 03_road.md

## 一、核心差異

| 面向 | 監造計畫 | 品質計畫 |
|------|---------|------------|
| 撰寫立場 | 監造單位（外部監督者） | 承攬廠商（施工自主管理者） |
| 目的 | 規定「如何監督廠商」 | 規定「如何自主管理品質」 |
| 主詞 | 監造單位應…、本公司審查… | 本公司（承攬廠商）應…、自主執行… |
| 查驗性質 | 抽查（外部稽查） | 自主檢查（內部管制） |
| 不符合處理 | 通知廠商限期改善 | 自行辦理矯正預防 |
