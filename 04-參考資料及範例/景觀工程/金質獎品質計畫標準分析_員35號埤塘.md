## 壹、員35號埤塘品質計畫的獨特特色

---

### 特色一：章節架構

> ⚠️ **兩件金質獎景觀計畫的章節數不同**：
> - 員35號埤塘（本件）：**8章（壹～捌）**，無設備功能運轉章
> - 平鎮運動公園（第一件）：**9章（壹～玖）**，有陸、設備功能運轉檢測程序及標準
>
> 差異原因：平鎮工程有複雜的機電系統（噴灌、照明監控、排水設備等），需要獨立的設備功能運轉測試章節；員35工程規模較小，無此需求。
> 製作時應依本工程是否有設備功能運轉需求，決定採8章或9章。

### 特色二：自主檢查表欄位多「量化檢查數據」欄

員35號埤塘的自主檢查表有**4欄格式**：

```
檢查項目 | 設計圖說、規範之檢查標準（定量/定性）| 實際檢查情形（量化檢查數據）| 檢查結果
```

★ **「量化檢查數據」欄是關鍵差異**：要求填入實際測量值，不只是打○╳。

例如：
- 混凝土工程：「坍度：15±4.0cm」→ 實際填入量測值（cm）
- 模板工程：「組模完成寬度：設計＿cm」→ 實際填入「＿cm」
- 鋼線網工程：「搭接長度≧15cm」→ 實際填入測量值

備註固定格式：
```
1.檢查標準及實際檢查情形應具體明確（例：磚砌完成後須不透光）
  或量化尺寸（例：磚縫7mm~10mm）。
```

---

### 特色三：品質管理標準表同時標示兩種停留點

與平鎮版本相同，同時標示：
```
＊為檢驗停留點
※為自主檢查停留點
```

---

### 特色四：鋼筋工程自主檢查表直接填入各構件鋼筋規格

一般計畫：只寫「符合設計圖說」
員35號埤塘：**直接在表格中列出各構件的鋼筋規格**，例如：

```
景觀平台基礎：8-D16、D16@15cm、(7支@1m)
景觀護欄基礎：D13@15cm、(7支@1m)
地景座階：10-D13、D13@20cm、(6支@1m)
S型溝蓋板：D13@13cm、(8支@1m)...
```

這讓現場人員不需翻設計圖說就能直接核對，大幅提升實用性。

---

### 特色五：施工流程圖（第五章）格式更清晰

與平鎮版本相同的三欄格式：
```
作業流程圖（含不合格退回箭頭）| 檢驗要點 | 相關記錄/文件
```

每個工項流程圖都清楚標示：
- ※自主查驗停留點
- ＊監造查驗停留點
- 不合格→退貨/退回修改

## 參、兩件金質獎計畫共同確立的「黃金標準」

以下是兩件金質獎計畫均共通的元素：

### 黃金標準一：章節結構
- 採**中文章次**（壹貳參…），不用數字
- **8章**（無設備功能運轉需求時）或 **9章**（有設備功能運轉需求時）
- 有機電設備（噴灌/照明監控/排水設備）→ 加入「陸、設備功能運轉檢測程序及標準」→ 共9章
- 無機電設備 → 不含此章 → 共8章
- **不含**獨立的矯正預防章、稽核章（整合入其他章節）

### 黃金標準二：品質管理標準表格式
- 雙停留點標記（＊監造停留點＋※自主檢查停留點）
- 每張表底部統一標示：「＊為檢驗停留點 ※為自主檢查停留點」
- 管理標準值量化（數字而非文字）

### 黃金標準三：施工流程圖格式
- 三欄式：作業流程圖｜檢驗要點｜相關記錄/文件
- 流程圖中標示不合格退回路徑
- ※和＊雙停留點標記

### 黃金標準四：自主檢查表固定底部格式
```
缺失複查結果：
□已完成改善、□未完成改善，填至「不合格管制總表」進行追蹤改善
複查日期：   年  月  日　複查人員職稱：       簽名：

備註：
1.檢查標準及實際檢查情形應具體明確或量化尺寸。
2.合格「○」，不合格「╳」，不需檢查「／」。
3.嚴重缺失未能及時改善，應填具「不合格品管制總表」追蹤改善。
4.本表由工地現場工程師或領班實地檢查後覈實記載簽認。

現場工程師：____________　　工地負責人：____________
```

★ 簽名欄為「現場工程師＋工地負責人」（非「現場施工人員＋工地主任」）

### 黃金標準五：文件編碼系統
- 每份文件有獨立分類代碼（文件紀錄分類代碼表）
- 附文件編碼圖例說明

## 肆、員35號埤塘品質計畫的3個加分亮點

### 亮點一：品質計畫書審查表裝訂於首頁
送審核章表→整體品質計畫書審查表→目錄
（比一般計畫多一份「審查表」，讓監造單位審查有具體依據）

### 亮點二：自主檢查表的「量化檢查數據」欄
要求填入實際測量值，而非只打○╳，更能客觀記錄品質狀況。

### 亮點三：鋼筋工程直接列入各構件規格
節省現場查圖時間，防止施工錯誤，是實務性最強的設計。

## 伍、綜合建議：製作金質獎水準計畫的完整檢核清單

### 架構面
- [ ] 採壹貳參…中文章次
- [ ] 依本工程是否有機電設備決定章節數：無設備→8章；有設備→9章（加陸、設備功能運轉）
- [ ] 自主檢查表位置：8章制在第陸章；9章制在第柒章
- [ ] 不含獨立矯正預防章、稽核章
- [ ] 計畫書首頁附品質計畫書審查表

### 品質管理標準面
- [ ] 每張表雙停留點標記（＊＋※）
- [ ] 管理標準值量化（具體數字）
- [ ] 混凝土：坍度15±4cm、溫度13～32°C、氯離子≦0.15kg/m³
- [ ] 鋼筋：搭接長度40D、保護層具體數值

### 施工流程圖面
- [ ] 三欄式格式（流程/要點/記錄）
- [ ] 流程圖含不合格退回路徑
- [ ] ※和＊雙停留點標記於流程圖中

### 自主檢查表面
- [ ] 採4欄格式（含量化數據欄）
- [ ] 有可勾選的□格式（非純文字）
- [ ] 鋼筋規格直接列入各構件（不只寫「依設計圖說」）
- [ ] 底部固定格式：複查欄＋備註4條＋「現場工程師＋工地負責人」雙簽名

### 文件管理面
- [ ] 建立文件分類代碼系統（附圖例）
- [ ] 末尾附人員證書（主任技師/品管/安衛）
