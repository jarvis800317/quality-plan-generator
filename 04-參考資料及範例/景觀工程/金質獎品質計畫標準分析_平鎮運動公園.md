## 壹、金質獎計畫 vs 一般計畫：7大關鍵差異

---

### 差異一：章節架構採「壹貳參」中文編號，非數字章

| 一般計畫 | 金質獎計畫 |
|---------|---------|
| 第一章 計畫範圍 | **壹、計畫範圍** |
| 第二章 管理責任 | **貳、管理責任及分工** |
| 第三章 施工要領 | **參、施工要領** |
| 第四章 品質管理標準 | **肆、品質管理標準** |
| 第五章 材料及施工檢驗程序 | **伍、材料與設備及施工檢驗程序** |
| 第六章 設備功能運轉測試 | **陸、設備功能運轉檢測程序及標準** |
| 第七章 自主檢查表 | **柒、自主檢查表** |
| 第八章 不合格品 | **捌、不合格品之管制** |
| （無）| **（無第九章矯正預防、第十章稽核）** |
| 第九章～十一章 | **玖、文件記錄管理系統** |

★ 金質獎計畫只有**9章**（去掉矯正預防與內部稽核獨立章，整合進其他章節）

---

### 差異二：名詞定義大幅擴充（27項，且有重要新增項目）

一般計畫：28項（以廠商立場常用詞彙為主）
金質獎計畫：27項，但**內容更精準**，特別新增：

- **十四、檢驗停留點（限止點）**：定義更詳細，說明「隱蔽部份於掩蓋前、永久性工程建造前」
- **十五、檢驗程序**：明確說明「書面文件依據」
- **廿一、矯正與預防措施**：納入名詞定義，而非只當章節標題
- **廿七、簽證**：明確列出所有應簽署文件範圍（施工計畫、施工圖、竣工圖、日報表等）

---

### 差異三：品質管理標準表新增「※自主檢查停留點」標記

一般計畫：只標示「＊檢驗停留點」
金質獎計畫：**同時標示兩種停留點**

```
＊為檢驗停留點（監造單位到場）
※為自主檢查停留點（廠商自主查驗）
```

每張品質管理標準表的管理標準欄位中，同時標記「施工前＊※」、「組立後＊※」，讓品管人員清楚區分哪個點要監造到場、哪個點自主查驗即可。

---

### 差異四：品質管理標準表內容精細量化

一般計畫品質管理標準：檢查標準多為文字敘述
金質獎計畫：**大量量化具體數值**，例如：

**混凝土工程標準表（金質獎版）：**
- 坍度：15±4 公分
- 混凝土溫度：13°C～32°C
- 氯離子含量：≦0.15Kg/m³
- 試體：每100m³製作一組，不足100m³仍需製作一組
- 澆置間隔：≦45 min
- 振動器：澆置15分鐘內振動，間隔45cm，時間5-10秒
- 荷載：澆置後24小時內禁止荷載
- 裂縫標準：裂縫寬不超過2mm（約拾圓硬幣厚度）

**鋼筋工程標準表（金質獎版）：**
- 保護層：與水土接觸75mm；不接觸50mm
- 鋼筋隔墊：#3間距≦60cm、#4≦80cm、#5以上≦100cm
- 搭接：40D，相鄰兩根錯開直徑25倍
- 焊接鋼線網搭接：>20cm

---

### 差異五：施工要領、品質管理標準、自主檢查表三者完整對應

一般計畫：三種表格分散在不同章節，工項數量可能不一致
金質獎計畫：**三種表格工項數量完全一致、編號對應**

例如：
- 表3-22「沙坑遊戲場工程施工作業要領」
- 表4-22「沙坑遊戲場工程施工品質管理標準」
- 表7-22「沙坑遊戲場工程施工自主檢查表」

三張表工項完全相同，構成「施工要領→管理標準→自主檢查」的完整品質迴路。

**共34工項（金質獎比一般計畫多許多特殊工項）：**
假設工程（告示牌/圍籬/施工架/放樣）、鋼筋/模板/混凝土/無細粒料混凝土鋪面、木作/鋼構、鋪面收邊/飾面/停車場/球場、座椅/牌示/洗手台/護欄/賣店/體健設施、擺盪遊戲場/沙坑遊戲場/地形遊戲場/挑戰遊戲場/陸域探索區/水域探索區、照明監控/排水設備/給水澆灌設備、植栽移植/新植喬木/新植灌木/草皮舖植、廁所工程

---

### 差異六：第五章施工檢驗程序採「作業流程及檢驗圖」格式

一般計畫第五章：施工檢驗程序表（文字條列式）
金質獎計畫第五章：**圖5-7-X 施工作業流程及檢驗圖**

每個工項有一張流程圖，包含三欄：
```
作業流程圖（左）| 檢驗要點（中）| 相關記錄/文件（右）
```

流程圖中標示：
- 不合格→退回重做箭頭
- ※自主查驗停留點
- ＊檢驗停留點

這種圖式格式比純文字表格更直觀，是金質獎計畫視覺品質的重要加分項。

---

### 差異七：工程概要含「本次變更數量」欄位

一般計畫表一-1：項次｜項目及說明｜單位｜數量（3欄）
金質獎計畫：**4欄**

```
項次 ｜ 項目及說明 ｜ 單位 ｜ 原契約數量 ｜ 本次變更數量
```

這反映實際施工管理現況，工程變更後原數量與變更數量並列，管理更嚴謹。

## 貳、金質獎計畫獨有的特色元素

### 1. 材料送審管制表更細緻
金質獎的表5-1「材料/設備送審管制總表」，針對每種材料列出：
- 送審資料清單（型錄、試驗報告、樣品、廠商資料等）
- 廠驗需求
- 預定/實際送審日期
- 審查意見欄位

### 2. 設備功能運轉章節有完整的給排水系統流程圖
金質獎計畫第六章（陸）有：
- 圖6-1 整體功能試運轉檢測程序
- 圖6-2 設備品管流程
- 圖6-3 給排水系統設備功能運轉檢驗流程圖
- 圖6-4 照明燈具系統設備功能運轉檢驗標準及流程圖
- 表6-1 機電設備功能運轉檢測標準一覽表
- 表6-2 噴灌設備功能運轉檢測標準一覽表
- 表6-3 機電設備功能運轉檢驗表
- 表6-4 給水設備功能運轉檢驗表

### 3. 文件分類代碼系統
金質獎計畫第九章設計了文件分類代碼系統（表9-1），每份文件有獨立編碼，並附圖9-1「文件編碼圖例」說明編碼規則。

### 4. 附件：人員證書+品管人員登錄表
金質獎計畫末尾附上：
- (一) 主任技師證書
- (二) 工地負責人回訓證書
- (三) 品管人員回訓證書
- (四) 職業安全衛生業務主管證書
- 品管人員登錄表

### 5. 品質管理標準表格腳統一標示
每張品質管理標準表底部都有統一腳注：
```
＊為檢驗停留點
※為自主檢查停留點
```

## 參、金質獎計畫關鍵提升點清單

製作品質計畫時，以下項目可向金質獎水準靠攏：

| 項目 | 一般計畫做法 | 金質獎做法 | 效果 |
|-----|-----------|---------|------|
| 章節編號 | 數字章 | 壹貳參中文 | 專業感提升 |
| 停留點標記 | 只有＊ | ＊和※雙標記 | 管制更清晰 |
| 品質標準值 | 文字敘述 | 具體數值量化 | 操作性更強 |
| 施工檢驗程序 | 文字表格 | 流程圖（三欄式） | 視覺清晰 |
| 工項對應 | 三章節可能不一致 | 三章節工項一一對應 | 品質迴路完整 |
| 施工數量表 | 原契約數量 | 含變更數量欄 | 反映實況 |
| 人員附件 | 無 | 附證書影本 | 符合法規要求 |
| 文件編碼 | 無系統 | 有代碼系統 | 管理有序 |

## 肆、使用本文件製作品質計畫的指引

1. **章節架構**：改用壹貳參中文編號
2. **停留點**：品質管理標準表每個檢查時機同時標＊和※
3. **量化數值**：混凝土工程至少包含坍度、溫度、氯離子、振動器間距等具體值
4. **施工檢驗程序**：以三欄流程圖格式呈現（作業流程圖｜檢驗要點｜相關記錄/文件）
5. **工項數量**：確保施工要領、品質管理標準、自主檢查表三者工項完全一致
6. **工程數量表**：施工項目及數量表加入「本次變更數量」欄
7. **末尾附件**：加入主任技師、品管人員、安衛人員證書及品管登錄表
8. **文件編碼**：建立文件分類代碼系統（可參照金質獎計畫格式）
