## 壹、道路工程金質獎計畫的架構

### 章節架構：11章制（與景觀工程8章完全不同）

| 章次 | 內容 |

## 貳、道路工程金質獎計畫的7大特色亮點

---

### 亮點一：自主檢查表使用「☆」標示檢驗停留點

景觀工程用「＊」和「※」雙符號標示
道路工程用**「☆」**標示檢驗停留點於自主檢查表格內的檢查項目

```
備註第4條：「☆」表示檢驗停留點之檢驗項目。
```

凡自主檢查表中需向監造單位申請會同查驗的項目，在「檢查項目」欄左側加「☆」。

---

### 亮點二：自主檢查表缺失複查格，加「檢附改善前中後照片」

一般景觀工程缺失複查格：
```
□已完成改善
□未完成改善，填至「不合格管制總表」...
```

道路工程金質獎版（更嚴格）：
```
□已完成改善（檢附改善前中後照片）
□未完成改善，填具「缺失改善追蹤表」進行追蹤改善
```

→ 要求完成改善後須附**改善前/中/後三階段照片**，且不合格追蹤用「缺失改善追蹤表」（名稱與景觀不同）

---

### 亮點三：材料自主檢查表（第五章）有具體尺寸規格

道路工程第五章20份材料自主檢查表（EM-XX），每種材料直接列出：
- 尺寸公差（具體公差數值）
- 強度標準（量化數值）
- 試體取樣頻率（多少m2/T取樣一次）

例如，預鑄路緣石材料自主檢查表（EM-05）：
```
高：27公分±3mm
長：60公分±2mm
上寬：15公分±2mm
下寬：20公分±2mm
破損大於1公分退回
```

例如，混凝土材料品質管理標準表（表五-2-22）：
```
坍度：15±3.5公分（注意：道路工程±3.5，景觀工程±4）
氯離子：<0.15 kg/m³
運送時間：出預拌廠至澆置完成不超過90分鐘
每批20~100m³取樣一組，超過100m³每100m³加取一組，一組五個試體
```

---

### 亮點四：品質管理標準表欄位採「流程卡式」格式（道路特有）

道路工程第五章施工檢驗程序表，採用獨特的**流程卡式**格式：

```
作業流程圖        |  檢查要點              |  相關記錄/文件
-----------------|-----------------------|----------------
準備工作          | 1.材料商之選擇          | 施工說明書
                 | 2.說明材料施工與檢驗規定  |
材料資料提送審查    | 1.材料生產或供應廠商文件  | 材料文件送審
                 | 2.施工機具技術資料       | 材料品質檢驗報告
※材料進場         | 1.出廠證明型錄試驗報告   | 現場檢驗報告
◎安全衛生查驗點    | 2.尺寸與圖說核對        | 強度試驗報告
```

注意兩種標記：
- **※** = 檢驗停留點（材料進場需申請）
- **◎** = 安全衛生查驗點（安衛相關）

→ 這是道路工程特有的**安全衛生查驗點「◎」標記**，景觀工程沒有。

### 亮點七：第十章內部品質稽核（景觀工程沒有）

道路工程品質計畫有**獨立的第十章：內部品質稽核**，包含5張表單：
- 表十-1 定期內部品質稽核計畫表
- 表十-2 內部品質稽核通知單
- 表十-3 內部品質稽核檢查表
- 表十-4 內部品質稽核改善通知單
- 表十-5 內部品質稽核成效追蹤表

景觀金質獎（8章制）沒有獨立稽核章，道路工程（11章制）有完整稽核制度。

## 肆、道路 vs 景觀：金質獎製作標準對照

### 兩類工程都要做的「共同黃金標準」

| 項目 | 標準 |
|-----|------|
| 計畫書首頁 | 附整體品質計畫書送審核章表 |
| 混凝土坍度 | 15±3.5cm（道路）或 15±4cm（景觀） |
| 氯離子 | ≦0.15 kg/m³ |
| 試體 | 每20~100m³取1組，超過100m³每增加100m³加取1組 |
| 文件編碼 | 建立文件分類代碼系統 |
| 施工流程圖 | 三欄式（流程/要點/記錄） |
| 自主檢查表底部 | 固定4條備註＋雙簽名欄 |
| 停留點標記 | 自主查驗停留點（景觀：※；道路：☆） |

### 道路工程獨有的金質獎加分項

| 項目 | 說明 |
|-----|------|
| 安衛查驗點「◎」 | 流程圖中標示安衛重要查驗節點 |
| 自主檢查表3分頁 | 施工前/中/後各一張 |
| 材料自主檢查表 | 第五章20份，每種材料一份（EM-XX） |
| 改善前中後照片 | 缺失改善複查格要求附三階段照片 |
| 第九章矯正預防 | 獨立成章，含流程圖2張＋表格2張 |
| 第十章內部稽核 | 獨立成章，含5張完整表單 |
| 施工日誌 | 表二-3，道路工程第二章特有 |
